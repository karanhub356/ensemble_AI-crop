# ensemble_AI-crop
Ensemble AI-based crop recommendation system using 6 ML models, GridSearchCV optimization, Top-3 probabilistic output, and DiCE counterfactual XAI. Achieved 99.55% accuracy on Kaggle Crop Dataset.

## Setup
pip install dice-ml scikit-learn pandas numpy matplotlib seaborn

## Dataset
Download Crop_recommendation.csv from:
https://www.kaggle.com/datasets/atharvaingle/crop-recommendation-dataset

## Run
python crop_framework_final.py# Hybrid AI-Based Sustainable Crop Planning Framework
